If you just want to install Stimfit and/or the stfio module, download and install MacPorts from here, then type into a terminal:

sudo port install stimfit py27-stfio py34-stfio

Building Stimfit from scratch requires you to install Xcode, MacPorts and a couple of libraries. Note that this may take several hours. There is experimental support for Stimfit in Homebrew, but currently it does not install a full version of Stimfit.
